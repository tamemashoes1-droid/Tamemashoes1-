# 🤖 بوت تميمة شوز - تليجرام

بوت تليجرام لخدمة عملاء متجر "تميمة شوز" - شباشب جملة.

## المميزات
- ✅ قائمة أزرار جاهزة (منتجات، أسعار، توصيل، تواصل)
- ✅ استقبال طلبات بخطوات (منتج → كمية → تليفون → عنوان)
- ✅ إشعار فوري للأدمن على تليجرام عند أي طلب جديد
- ✅ حفظ كل الطلبات في ملف `orders.json`
- ✅ ردود ذكية على أي سؤال حر (اختياري - يحتاج Anthropic API Key)

---

## 🚀 خطوات التشغيل على Render (مجاني)

### 1️⃣ أنشئ حساب
روح على [render.com](https://render.com) واعمل حساب جديد (تقدر تسجل بإيميلك أو بحساب جيميل).

### 2️⃣ رفع الكود على GitHub
لازم الكود يكون في مستودع (repository) على GitHub أولاً:
1. اعمل حساب على [github.com](https://github.com) لو معندكش
2. اعمل **New repository** باسم `tameema-bot`
3. ارفع الملفات الموجودة هنا (`bot.js`, `package.json`, `products.json`, `orders.json`) فيه
   - أسهل طريقة: من موقع GitHub نفسه اضغط "Add file" → "Upload files" واسحب الملفات

### 3️⃣ إنشاء Web Service على Render
1. من لوحة تحكم Render اضغط **New** → **Background Worker** (أو Web Service)
2. اربطه بحساب GitHub بتاعك واختار المستودع `tameema-bot`
3. في الإعدادات:
   - **Build Command:** `npm install`
   - **Start Command:** `npm start`
4. روح لقسم **Environment Variables** وضيف:
   - `BOT_TOKEN` = التوكن بتاع البوت (اللي جبته من BotFather)
   - `ADMIN_CHAT_ID` = آيدي تليجرام بتاعك (شرح تحت إزاي تجيبه)
   - `ANTHROPIC_API_KEY` = (اختياري) مفتاح Anthropic لو عايز الردود الذكية تشتغل
5. اضغط **Create Background Worker** أو **Deploy**

### 4️⃣ إزاي تجيب ADMIN_CHAT_ID بتاعك
1. روح على تليجرام وابحث عن بوت اسمه `@userinfobot`
2. اضغط Start
3. هيديك رقم (Chat ID) - ده اللي تستخدمه

## 🔧 تعديل المنتجات
المنتجات في ملف `products.json` - افتحه وعدّل الأسماء والأسعار والكميات الأساسية حسب منتجاتك الحقيقية. الشكل:
```json
{ "name": "اسم المنتج", "price": 85, "moq": "12 جوز/كرتونة" }
```

بعد التعديل، لازم ترفع الملف المعدّل تاني على GitHub، وRender هيحدث البوت تلقائياً.

## 📋 أوامر البوت
- `/start` - بدء المحادثة
- `/menu` - عرض القائمة الرئيسية
- `/products` - عرض المنتجات
- `/order` - بدء عملية طلب جديد

---

## ⚠️ ملاحظات أمان
- لا تشارك `BOT_TOKEN` مع أي حد
- لو حسيت إن التوكن اتسرب، روح BotFather واعمل `/revoke` فوراً
- ملف `orders.json` بيتصفر مع كل إعادة تشغيل على الباقة المجانية من Render (لأن التخزين مؤقت) - لو عايز حفظ دائم للطلبات قولي وأضيف Firebase
